# mmd-lwjgl

A Kotlin application to load pmd files with LWJGL

## Usage

* Download "HatsuneMiku.pmd" into `src/main/resources` or `src/test/resources`
* Execute following commands:

```
$ ./gradlew build -x test
$ ./gradlew run
```
